Fobj_MDP {BRKGA}R Documentation

Objective function of the maximum diversity problem

Description

Calculates the sum of all distances associated with M

Usage

Fobj_MDP(D, Xm)

Arguments

D

distance matrix

Xm

elements of M

Details

Fobj_MDP

Value

SD

= sum all distances associated with M
