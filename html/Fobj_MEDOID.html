Fobj_MEDOID {BRKGA}R Documentation

Objective function of the k-medoids problem

Description

Computes the sum of the distances for each object from near object associated with medoid

Usage

Fobj_MEDOID(D, x, k)

Arguments

D

distance matrix

x

medoids

k

number of clusters (is necessary to define Exa2=k)

Details

Fobj_MEDOID

Value

S

= sum of distances

References

1. Kaufman L. e Rousseeuw P.J. (1989). Finding Groups in Data. An Introduction to Cluster Analysis. Wiley-Interscience Publication.

2. Zhang, Q. and Couloigner, I. (2005). A New Efficient k-medoid Algorithm for Spatial Clustering. Lecture Notes in Computer Science, v3482, 181-189.

Examples

D<-as.matrix(dist(iris[,1:4]))
N<-nrow(iris)
k<-3 #clusters
x<-sample(N,k)  #selected medoids
fobj<-Fobj_MEDOID(D,x,k)
