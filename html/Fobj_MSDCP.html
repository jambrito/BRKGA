Fobj_MSDCP {BRKGA}R Documentation

Objective function of the minimum sum distance clustering problem

Description

Fobj_MSDCP

Usage

Fobj_MSDCP(D, cluster, k)

Arguments

D

distance Matrix

cluster

clustering Vector

k

number of clusters (is necessary to define Exa2=k)

Value

SD

= sum of distances

References

1. Friedman, J. H. and Meulman J.J. (2004). Clustering objects on subsets of attributes. #' Journal Royal Statistics Society B, Part 4,66:815-849.

2. Hansen, P. and Jaumard, B. (1997). Cluster Analysis and Mathematical Programming. #'Mathematical Programming, 79:191-215.

3. Rao, M.R. (1971). Cluster Analysis and Mathematical Programming. Journal of American Statistical Association, 66:622-626.

Examples

k<-3 #Clusters
Distance<-as.matrix(dist(iris[,1:4]))
N<-nrow(iris)
u<-popgen(n=N,p=1)
cluster<-Decoder_MSDCP(u,k)
SD<-Fobj_MSDCP(Distance,cluster,k)
