Decoder_MEDOID {BRKGA}R Documentation

Decoder implemented to k-medoids Problem

Description

Determines k objects associated with medoids

Usage

Decoder_MEDOID(u, k)

Arguments

u

chromossome

k

number of clusters (is necessary to define Exa1 = k)

Details

Decoder_MEDOID

k-medoids problem is fully described in Kaufman and Rousseeuw (1989)

Value

medoids

= vector with k-medoids (objects)

References

1. Kaufman L. e Rousseeuw P.J. (1989). Finding Groups in Data. An Introduction to Cluster Analysis. Wiley-Interscience Publication.

2. Zhang, Q. and Couloigner, I. (2005). A New Efficient k-medoid Algorithm for Spatial Clustering. Lecture Notes in Computer Science, v3482, 181-189.

Examples

N<-nrow(iris)
u<-popgen(n=N,p=1)
k<-3
medoids<-Decoder_MEDOID(u,k)
